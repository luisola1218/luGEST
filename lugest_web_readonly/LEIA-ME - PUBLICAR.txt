PUBLICAR NO CPANEL

1. Criar subdomínio, por exemplo:
   consultas.teudominio.pt

2. Fazer upload da pasta lugest_web_readonly

3. Apontar o document root para:
   lugest_web_readonly/public

4. Na pasta config:
   - copiar config.cpanel.example.php
   - renomear para config.php
   - ajustar dados MySQL

5. Criar utilizador MySQL só com leitura:
   ver sql/lugest_web_readonly_user.sql

6. Opcional:
   executar sql/lugest_web_readonly_views.sql

7. Testar login no domínio

Notas:
- o portal é apenas de consulta
- não faz INSERT, UPDATE nem DELETE
- deve usar utilizador MySQL exclusivo com SELECT
